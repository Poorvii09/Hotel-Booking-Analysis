# 🌿 Hotel Booking Analysis - Data Analytics Project

## 📌 Project Overview

This project analyzes booking and operational data from Shelar Agro Tourism to uncover business insights using Python, SQL, and Power BI.

The goal is to transform raw booking records into interactive dashboards that support better business decisions.

---

## 🚀 Tech Stack

- Python (Pandas)
- MySQL
- Power BI
- Excel
